# Contributing to Matrix IPTV CLI

First off, thank you for considering contributing to Matrix IPTV CLI! It's people like you who make it a great tool for the community.

## How Can I Contribute?

### Reporting Bugs

- Check the existing issues to see if the bug has already been reported.
- If not, open a new issue. Include as much detail as possible, such as your OS, steps to reproduce, and any error logs.

### Suggesting Enhancements

- Open a new issue and describe the feature you'd like to see.
- Explain why it would be useful.

### Pull Requests

- Fork the repo.
- Create a new branch for your feature or bug fix.
- Write your code and ensure it follows the project's style (run `cargo fmt`).
- Ensure the project compiles without warnings (`cargo check --all-targets --all-features`).
- Submit a pull request.

## Development Setup

1.  **Prerequisites**: You'll need Rust and Cargo installed.
2.  **Clone the Repo**: `git clone https://github.com/yourusername/matrix-iptv.git`
3.  **Build**: `cargo build`
4.  **Run**: `cargo run`

## Code Style

We follow standard Rust formatting. Please run `cargo fmt` before submitting your PR.
